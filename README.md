# imagemin-watch

Watch changes and compress images

## Usage

### CLI

Compress all png/jpg/jpeg

```
yarn cli compress <source> <dest>
yarn cli compress --after <time> <source> <dest>
yarn cli compress --before <time> <source> <dest>
```

Watch changes

```
yarn cli watch <source> <dest>
```
